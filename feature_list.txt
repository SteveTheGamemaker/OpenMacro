OTHER SYSTEMS
=============

Variables:
- Types: boolean, integer, decimal, string, dictionary, array
- User-defined and system-defined
- Local variables (scoped to action blocks)
- Global variables (accessible across all macros)

Magic Text (template variables):
- Device info: {battery_temp}, {battery_current_now}, {android_version_sdk}, {ram_total}, {ram_available}, {screen_resolution}, {storage_free_internal}, {storage_free_external}
- Connectivity: {ssid}, {wifi_strength}, {cell_connection_type}, {cell_id}, {ip_address}
- Time: {year}, {month}, {minute}, {second}, {system_time}, {system_time_ms}, {week_of_year}
- Communication: {call_name}, {call_number}, {sms_name}, {sms_number}, {sms_message}
- Notifications: {not_title}, {not_ticker}
- Other: {shared_text}, {ui_click_text}, {webhook_caller_ip_address}, {webhook_url}
- Variable references: {lv=variable_name}, dictionary access via {lv=dict[key][subkey]}

Action Blocks:
- Reusable groups of actions (like functions)
- Support input parameters and output parameters
- Local/working variables scoped within the block

Stopwatches:
- Named timers that can be started, stopped, reset
- Can be used as triggers or in constraints

Geofences:
- Named circular zones on a map
- Used with geofence triggers for enter/exit detection

Macro Structure:
- Each macro has: 1+ triggers, 1+ actions, 0+ constraints
- Multiple triggers are OR logic (any one fires the macro)
- Constraints are AND logic by default (all must be true)
- Actions execute sequentially by default
- Async option available for long-running actions

Community/Templates:
- Shareable macro templates
- Categories: top new, top rated, newest
- Import/export macros as files

Backup:
- Local automatic backups
- Cloud backups (pro feature)
- Export macros action for automated backups

Plugins:
- Third-party app integration
- Tasker/Locale plugin compatibility
- Plugin store for discovering compatible apps

Wear OS:
- Companion app for smartwatch integration
- Watch complications support

Accessibility Service:
- Used for UI interaction features (clicking, reading screen)
- Required for some advanced triggers like UI click and screen content

Root Features:
- Shell commands
- Toggle mobile data
- Some system settings require root or ADB hack
