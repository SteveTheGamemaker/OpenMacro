# OpenMacro

</div>

A free, open-source macro automation app for Android.

## What is OpenMacro?

OpenMacro lets you automate your Android device by creating **macros** — simple rules that say "when *this* happens, do *that*." No subscriptions, no paywalls, no ads. Just automation that works.

**Examples of what you can do:**
- When you connect to your home WiFi → turn volume up and disable Bluetooth
- Every weekday at 8am → enable Do Not Disturb and launch your work app
- When battery drops below 20% → turn off WiFi and reduce brightness
- When you receive an SMS from a specific contact → play a custom notification sound

## Features

Each macro is built from three parts:

- **Triggers** — events that start a macro (time, battery level, WiFi connection, app launch, etc.)
- **Actions** — things to do when a trigger fires (send notification, change volume, launch app, etc.)
- **Constraints** — conditions that must be true for the macro to run (only on weekdays, only when charging, etc.)

OpenMacro is under active development. See [feature_list.txt](feature_list.txt) for the full roadmap.

## Building

1. Clone the repo
2. Open in Android Studio
3. Sync Gradle and run on a device or emulator (API 28+)
